# linxxery.com

浙江省立体地形图 · Zhejiang 3D Topographic Map

Interactive 3D terrain visualization featuring:
- SRTM elevation data with custom terrain colormap
- Provincial administrative boundaries
- 11 prefecture-level cities + landmarks
- Major river systems (钱塘江, 瓯江, etc.)
- Terrain contours & labels

Built with Python, Plotly, and open geospatial data.

## Data Sources
- Elevation: NASA SRTM (Shuttle Radar Topography Mission)
- Boundaries: GADM (Database of Global Administrative Areas)
- Cities & geography: Custom compilation
